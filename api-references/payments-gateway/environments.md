# Environments

URLs

* **Sandbox**: `https://sandbox-gw.simpleps.com`
* **Production**: `https://gw.simpleps.com`

Tokenizer:

* **Sandbox:**  `https://cdn.simpleps.com/sandbox/inyo.js`
* **Production:** `https://cdn.simpleps.com/production/inyo.js`

#### Use Cases

1. **Sandbox Environment**:
   * Use this environment for development and testing purposes.
   * Simulate real-world scenarios without affecting live data or incurring financial risks.
2. **Production Environment**:
   * Use this environment for live, real-money transactions.
   * Ensure all configurations and integrations are thoroughly tested before deployment to production.
